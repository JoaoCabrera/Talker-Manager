## Para rodar o projeto:

É necessario ter o Git e Node instalados para rodar o projeto.

## Clone o projeto:

`git clone git@github.com:JoaoCabrera/Talker-Manager.git`

## Instale as dependências:

`npm install`

## Inicie o projeto:

`npm start`
